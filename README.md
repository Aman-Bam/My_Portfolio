## 🛠 Technical Architecture

This isn't just a static page; it's a curated experience built with a modern frontend stack focused on fluidity and performance.

### Core Stack
- **Frontend:** React 19 (Concurrent Mode), TypeScript
- **Styling:** Tailwind CSS 4.0 (Next-gen engine)
- **Animations:** GSAP (ScrollTrigger) & Framer Motion
- **Build Tool:** Vite
- **package Manager:** npm
- **UI Components:** Radix UI & Lucide Icons
- **Asset Management:** ImageKit Integration

### Key Engineering Features
- **Immersive Motion:** Implementation of a custom **Magnetic Cursor** and smooth-scroll physics.
- **Horizontal Storytelling:** A pinned horizontal gallery for projects using GSAP `ScrollTrigger`, transforming the standard vertical scroll into an interactive experience.
- **Systems-First Design:** A terminal-inspired "About" section reflecting a developer's environment and a dynamic, animated experience timeline.
- **Performance Optimized:** Leveraging `memo` for expensive components and lazy-loading assets via ImageKit to ensure a 60FPS experience.

## ⚙️ Local Development

### Prerequisites
- Node.js (Latest LTS)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-portfolio.git
   cd your-portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory:
   ```env
   VITE_IK_URL_ENDPOINT=your_imagekit_endpoint
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```
